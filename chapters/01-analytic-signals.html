    A complex number is the sum of a real number and an imaginary number.
    It takes the form:

    $$ z \ = \ x + \sqrt{-1} \cdot y $$
    where $ x $ is the real part of $ z $:
    $$ \Re (z) \ = \ x $$
    and $ y $ is the imaginary part of $ z $:
    $$ \Im (z) \ = \ y $$

    Let's imagine a Cartesian plane whose horizontal axis (abscissa) is the set of real numbers, and whose vertical axis (ordinate) is the set pure imaginary numbers. This plane is called the complex plane.

    Now imagine we plot $ z $ on this plane. It shows how any complex number can be regarded as a vector joining the origin of both axis and the point with coordinates $ \ [x = \Re (z) \ , \ y = \Im (z)] \ $.

    This vector traces the diagonal of a rectangle with two sides $ \ = \Re (z) \ $ and two sides $ \ = \Im (z) \ $.
    The vector $ z $ splits this rectangle in two identical right triangles whose hypotenuse is the length (magnitude) of the vector.
    This hypotenuse is called the absolute value (or magnitude) of the complex number $ z $ and is notated $ \ \lvert z \rvert \ $ — sometimes $ \ \abs (z) \ $.

    The theorem of Pythagoras gives:

    $$ \lvert z \rvert \ = \ \sqrt{ \Re (z)^2 + \Im (z)^2 } $$

    Notice that if $ \ \Im (z) = 0 \ $ then $ \ \lvert z \rvert = \sqrt{\Re (z)^2} \ $ which results in ignoring $ z $'s sign. Which is often mistaken for the actual definition of a number's absolute value.
    It stops being valid once said number has an imaginary part, while the definition given by complex analysis applies to purely real numbers.
    This coincides with the fact that real numbers are a subset of complex numbers.

    Let's call $ \theta $ the angle formed by the vector $ z $ and the real (horizontal) axis.
    $ \theta $ is called the argument of the complex number $ z $ and is notated $ \ \arg (z) \ $.
    Since $ \ \Re (z) \ $ is its adjacent side and since $ \ \Im (z) \ $ is its opposite side:

    $ \theta $ (theta), not $ 0 $ (zero).

    $ \tan $$ (\theta) \ = \ \Im (z) / \Re (z) $

    And thus:

    $ \arg (z) \ = \ $$ \arctan $$ \bigl( \Im (z) / \Re (z) \bigr) $

    or rather:

    $ \atanTwo $$ \bigl( \Im (z) \ , \ \Re (z) \bigr) $

    atan2:

    The regular $ \arctan $ function — $ \ \atan (x) \ $, $ \ \tan^{-1} (x) \ $ — tends to $ - \pi / 2 $ when $ x $ tends to $ - \infty $;
    it tends to $ + \pi / 2 $ when $ x $ tends to $ + \infty $.
    It only outputs angles in the North-East and South-East quadrants.

    The argument of a complex number is in the range $ ] - \pi \ , \ + \pi ] $. It spans a full turn (4 quadrants):

    • South-West, $ ] - \pi \ , \ - \pi / 2 ] $:
      $ \Re (z) \ $ and $ \ \Im (z) \ $ are both negative. $ \ \Re (z) \ $ crosses zero at $ - \pi / 2 $.
    • South-East, $ ] - \pi / 2 \ , \ 0 ] $:
      $ \Re (z) \ $ is positive, $ \ \Im (z) \ $ is negative. $ \ \Im (z) \ $ crosses zero at zero.
    • North-East, $ ] 0 \ , \ + \pi / 2 ] $:
      $ \Re (z) \ $ and $ \ \Im (z) \ $ are both positive. $ \ \Re (z) \ $ crosses zero at $ + \pi / 2 $.
    • North-West, $ ] + \pi / 2 \ , \ + \pi ] $:
      $ \Re (z) \ $ is negative, $ \ \Im (z) \ $ is positive. $ \ \Im (z) \ $ crosses zero at $ + \pi $.

    If you feed the ratio directly to $ \arctan $, you've lost track of which term is responsible for the sign.
    All $ \arctan $ can possibly know is whether the denominator and the numerator have the same sign.
    By having two separate variables, $ \ \atanTwo (y , x) \ $ fixes that.

    E.g.:

    You feed your ratio to $ \arctan $ and get $ \pi / 3 $. (So North-East quadrant.)
    But you know that both terms were negative.
    So you know that you should be in the South-West quadrant.
    That's a 2-quadrant jump. So half a circle. So $ \pm \pi $.
    $ + \pi $ would outreach the boundaries $ ] - \pi \ , \ + \pi ] $:

    $$ \frac{\pi}{3} + \pi \ = \ \frac{\pi + 3 \pi}{3} \ = \ \frac{4}{3} \pi $$

    so you know the argument is:

    $$ \pi / 3 - \pi \ = \ -2 \pi / 3 $$

    Also: if $ \ x = 0 \ $, $ \ \arctan (y/x) \ $ is fed an exploding division by zero...
    $ \atanTwo (y , x) \ $ fixes that too.

    Remember that $ \adj $ is $ \Re (z) $, $ \opp $ is $ \Im (z) $, $ \hyp $ is $ \lvert z \rvert $ and $ \theta $ is $ \arg (z) $.
    Also that $ \ z \ = \ \Re (z) + \sqrt{-1} \cdot \Im (z) \ $.

    Then:

    $$ \begin{alignat*}{2} \Re (z) \ = \ & \cos \bigl( \arg (z) \bigr) \cdot \lvert z \rvert \\ \Im (z) \ = \ & \sin \bigl( \arg (z) \bigr) \cdot \lvert z \rvert \end{alignat*} $$

    Then:

    $$ z \ = \ \lvert z \rvert \cdot \Bigl( \ \cos \bigl( \arg (z) \bigr) \ + \ \sqrt{-1} \cdot \sin \bigl( \arg (z) \bigr) \ \Bigr) $$

    This expression is called the polar form of the complex number $ z $.

    Complex numbers can be regarded as vectors. Vectors also have a polar form.
    The notation: $ \ \mathbf{v} \ = \ ( \ r \ , \ \angle \theta \ ) $
    means: $ \ \mathbf{v} $ is a vector of magnitude $ r $ and angle $ \theta $.
    If $ \mathbf{v} $ is represented with the complex number $ z $, then: $ \ \lvert z \rvert = r \ $ and $ \ \arg (z) = \theta \ $.

    The absolute value $ \lvert z \rvert $ of $ z $ is sometimes also called modulus or magnitude.
    The argument $ \ \arg (z) \ $ of $ z $ is sometimes also called phase.

    02 — Euler's formula

    Let $ \ j = \sqrt{-1} \ $ and $ \ x \ $ be a real number, then Euler's formula states that:

    $ \exp $ $ (j x) \ = \ \cos (x) + j \sin (x) $

    ↑↑↑ Don't overlook that one. ↑↑↑
    (We'll demonstrate it at some point.)

    By making use of the 2 following trigonometric identities :

    $$ \cos (-x) \ = \ \cos (x) \qquad \text{and} \qquad \sin (-x) \ = \ - \sin (x) $$

    Euler's formula can be re-expressed as:

    $$ \exp (j \cdot -x) \ = \ \cos (-x) + j \sin (-x) \ = \ \cos (x) - j \sin (x) $$

    Then:

    $$ \exp (j x) - \exp (-j x) \ = \ \cos (x) + j \sin (x) - ( \cos (x) - j \sin (x) ) \ = \ 2 j \sin (x) $$

    And:

    $$ \exp (j x) + \exp (-j x) \ = \ \cos (x) + j \sin (x) + ( \cos (x) - j \sin (x) ) = 2 \cos (x) $$

    Solving [1] $$ \exp (j x) - \exp (-j x) \ = \ 2j \sin (x) $$ for $ \sin (x) $ gives:

    $$ \sin (x) \ = \ \frac{ \exp (j x) - \exp (-j x) }{2 j} $$

    Solving [2] $$ \exp (j x) + \exp (-j x) \ = \ 2 \cos (x) $$ for $ \cos (x) $ gives:

    $$ \cos (x) \ = \ \frac{ \exp (j x) + \exp (-j x) }{2} $$

    Considering the identity:

    $$ \cos ( \theta ) \ = \ \sin ( \theta + \pi / 2 ) $$

    equation [3] $$ \cos (x) \ = \ \frac{ \exp (j x) + \exp (-j x) }{2} $$ shows that any real sinusoid can be expressed as half the sum of two complex exponentials.
    This is interesting because the rules of exponentiation are a lot simpler than the rules of trigonometry.
    For instance, $ \ \exp (x) \ $ is its own derivative (and thus anti-derivative) — by definition.
    This makes processes such as integration or differentiation become very trivial, compared to the same processes applied to $ \ \cos (x) \ $ or $ \ \sin (x) \ $.
    (Actually, derivatives and antiderivatives are not where real trigonometric functions are the trickiest, but: $ \ \cos ( a x + b ) \ $, etc.)

    Let $ \quad z_1 \ = \ \exp (j x) / 2 \quad $, $ \quad z_2 \ = \ \exp (-j x) / 2 \quad $, then:

    $$ z_1 \ = \ \cos (x) / 2 \ + \ j \ \times \ + \sin (x) / 2 \\ z_2 \ = \ \cos (x) / 2 \ + \ j \ \times \ - \sin (x) / 2 $$

    Their respective real ($ \Re $) and imaginary ($ \Im $) parts are thus:

    $$ \begin{align} \Re ( z_1 ) \ =& \ \cos (x) / 2 &;&& \Im ( z_1 ) \ =& \ + \sin (x) / 2& \\ \Re ( z_2 ) \ =& \ \cos (x) / 2 &;&& \Im ( z_2 ) \ =& \ - \sin (x) / 2& \\ \\ \implies \ \ \Re ( z_1 ) \ =& \ \Re ( z_2 ) &\quad ;&& \quad \Im ( z_1 ) \ =& \ - \Im ( z_2 )& \end{align} $$

    $ z_1 $ and $ z_2 $ form a complex conjugate pair.
    (Which means: they have equal real parts and opposite imaginary parts.)
    So any real sinusoid (equation [3] $$ \cos (x) \ = \ \frac{ \exp (j x) + \exp (-j x) }{2} $$ ) can be expressed as the sum of two complex conjugates.

    Complex conjugates are usually denoted by a horizontal bar over the number's symbol: $ \ {z_2 \ = \ \overline{z_1}} \ $ means that $ z_2 $ and $ z_1 $ are conjugates.
    Sometimes also with an asterisk: $ \ {z_2 \ = \ z_1^*} $.
    First option is really not convenient because monospace (... GitHub's the original
    — you're reading the derivative...)     .
    Second option looks like multiplication in most programming languages.
    So I'll use $ \ {z_2 \ = \ \conj ( z_1 )} \ $ — the Desmos way, for practical reasons.
    But know that this is not standard and be ready to encounter other conventions.

    Euler's formula and the polar form

    Let's consider a complex number $ z $ with argument $ \arg (z) $ and absolute value $ \lvert z \rvert $.

    According to Euler's formula:

    $$ \exp \bigl( j \arg (z) \bigr) \ = \ \cos \bigl( \arg (z) \bigr) + j \sin \bigl( \arg (z) \bigr) $$

    Recalling the polar form expression:

    $$ z \ = \ \lvert z \rvert \cdot \Bigl( \ \cos \bigl( \arg (z) \bigr) + j \sin \bigl( \arg (z) \bigr) \ \Bigr) $$

    Combining these two expressions leads to:

    $$ z \ = \ \lvert z \rvert \exp \bigl( j \arg (z) \bigr) $$

    I haven't found any documentation on this form (perhaps from not digging deep enough...) so let's call it the exponential form of the complex number $ z $.

    This means that any complex number in the form:

    $$ \zeta \ = \ \alpha \exp ( j \theta ) $$

    has an absolute value:

    $$ \lvert \zeta \rvert \ = \ \alpha $$

    and an argument:

    $$ \arg ( \zeta ) \ = \ \theta $$

    Then:

    $$ \Re ( \zeta ) \ = \ \alpha \cos ( \theta ) $$

    and:

    $$ \Im ( \zeta ) \ = \ \alpha \sin ( \theta ) $$

    Now in the special case:

    $$ \theta \ = \ \omega t $$

    where $ t $ is the time (in seconds) and $ \omega $ is an angular frequency (in radians per second), the exponential form:

    $$ \zeta \ = \ \alpha \exp ( j \omega t ) $$

    is what is called a complex sinusoid — sometimes a phasor.
    It is also called an analytic signal as it has optimal qualities in order to study a system's response to pure sinusoids.
    Indeed comparing equations [3] $$ \cos (x) \ = \ \frac{ \exp (j x) + \exp (-j x) }{2} $$ and [4] $$ \zeta \ = \ \alpha \exp ( j \omega t ) $$ reveals that if we express a real sinusoid in terms of complex exponentials, we end up with two complex sinusoids, one of which has a negative frequency; while a pure complex sinusoid has its amplitude and phase embedded in its constant factor and only involves positive frequencies.
    More on that right away.

    03 — Amplitude, frequency & phase

    If $ t $ is the time (in seconds), $ \ f (t) \ = \ \cos (t) \ $ is a sinusoid of:

    Amplitude:
    $$ \alpha \ = \ 1 $$
    Period:
    $$ T \ = \ 2 \pi \ \text{ seconds} $$
    Frequency:
    $$ \f \ = \ 1 / 2 \pi \ \text{ Hertz (Hz)} $$
    Phase:
    $$ \varphi \ = \ 0 \ \text{ radian (rad)} $$

    That is, $ \ f (t) \ $ oscillates from $ -1 $ to $ +1 $ through a repetitive pattern of length $ = \ 2 \pi \ $ with an offset $ = \ 0 $.

    Notice that $ \ f (t) \ = \ \alpha \cos (2 \pi \f t + \varphi) $
    Or, after defining the angular frequency $ \ \omega \ = \ 2 \pi \f \ $ (rad/s):

    $$ f (t) \ = \ \alpha \cos (\omega t + \varphi) $$

    That is, $ \ f (t) \ $ oscillates from $ -\alpha $ to $ +\alpha $ through a repetitive pattern of length $ = 1 / \omega \ $ with an offset $ = \varphi $.

    Note that for $ \varphi $ to be expressed in radians (that is as a portion of the modulus $ 2 \pi $), the offset $ t_0 $ (seconds) must be defined.
    $ t_0 $ is the time $ \ f (t) \ $ takes to reach the beginning of an oscillation cycle ($ \ f (t) \ = \ {\alpha \cos (0)} \ = \ \alpha \ $) for the first time.
    (This implies $ \ \lvert t_0 \rvert \ \lt T \ $.)

    $$ \begin{align} \text{Then: } && \ f (t) \ = \ & \alpha \cos \bigl( \omega (t - t_0) \bigr) & \\ \text{And: } && \ \varphi \ = \ & \omega \ \cdot \ - t_0 & \end{align} $$

    Remember that $ \ \cos (\theta) \ = \ \sin (\theta + \pi / 2) \ $.
    Which means that the relationship between sine and cosine is simply a constant horizontal offset.
    I chose $ \ f (t) \ = \ \cos (t) \ $ as the base sinusoid, but $ \ f (t) \ = \ \sin (t) \ $ would be just as valid.
    In such a case $ \ f (t) \ = \ \cos (t + \varphi) \ $ would have a phase $ \ \varphi \ = \ \pi / 2 \ $ rad.

    This is a pure matter of convention.
    Since converting one convention into the other is as trivial as an addition, which one we choose has few consequences.
    So if one convention simplifies the maths, why choose the other? Right?

    We're just about to call back Euler's formula. Choosing cosine avoids having j in the denominator.
    I hope now it doesn't feel arbitrary anymore.

    Also: because $ \ sin (0) \ = \ 0 \ $, sine is inherently unable to express constants (with null frequencies);
    cosine, on the other hand, outputs 1 at 0 — which is quite convenient.

    Theoretically, sine would only need a quarter-circle of phase to match cosine,
    but this kind of collides with the expression of $ \varphi $: $$ -t_0 \cdot 0 \ = \ \pi/2 \quad \iff \quad t_0 \ = \ - \frac{\pi}{0 \cdot 2} $$ ... not super legal.
    Yet this undefined result is a useful information: the phase of a constant is an irrelevant concept.

    Using equation [3] $$ \cos (x) \ = \ \frac{ \exp (j x) + \exp (-j x) }{2} $$ to express $ \ f (t) \ $ in terms of complex exponentials instead of raw trigonometry:

    $$ f (t) \ = \ \frac{\alpha}{2} \exp \bigl( j (\omega t + \varphi) \bigr) + \frac{\alpha}{2} \exp \bigl( -j (\omega t + \varphi) \bigr) $$ $$ f (t) \ = \ \frac{\alpha \exp (j \varphi)}{2} \exp (j \omega) + \frac {\alpha \exp (j \ \cdot \ - \varphi)}{2} \exp (j \ \cdot \ - \omega t) $$

    Comparing this expression of $ \ f (t) \ $ to equation [4] $$ \zeta \ = \ \alpha \exp (j \omega t) $$ shows that a real sinusoid is the sum of two complex sinusoids, one of which has a negative frequency.

    Let's define:

    $$ s \ = \ j \omega $$

    and

    $$ X (s) \ = \ \frac{\alpha \exp (j \phi)}{2} $$

    $ X (s) \ $ is a complex number in exponential form with absolute value (magnitude) $ \ \lvert X (s) \rvert \ = \ \alpha / 2 \ $ and argument $ \ \arg \bigl( X (s) \bigr) \ = \ \varphi \ $. Then any complex sinusoid in the form:

    $$ f (t) \ = \ X (s) \exp (st) $$

    where $ s $ is a pure imaginary number! ( $ \Re (s) = 0 $ )

    can be considered the positive-frequency-component of a real sinusoid with:

    Amplitude:
    $$ \alpha \ = \ 2 \ \lvert X (s) \rvert $$
    Phase:
    $$ \varphi \ = \ \arg \bigl( X (s) \bigr) $$
    Frequency:
    $$ \omega \ = \ s / j \ \text{rad/s} \qquad \text{or} \qquad \f \ = \ s / 2 \pi j \quad \text{Hz} $$

    $ X (s) \ $ is called the complex amplitude of $ \ f (t) \ $. It embeds the real amplitude and the phase.
    $ X (s) \ $ is just a symbol. The complex sinusoid $ \ A \exp (z t) \ $ has a complex amplitude $ = A $ as long as $ \ \Re (z) \ = \ 0 \ $.
    (If $ A $ is purely real, then the phase is zero, and complex amplitude $ = $ real amplitude.)

    And thus the two (positive and negative) complex sinusoids that form a real sinusoid have equal amplitude and opposite phase.
    (Opposite phase because we measure it in radians, in fact they have the same $ t_0 $.)

    Infinite feedback loop:

    • A real sinusoid is a sum of complex sinusoids.
    • A complex sinusoid is the sum of a real and an imaginary sinusoid.

    There is a process called Hilbert transform which applies a different phase shift ($ \pm \pi / 2 $) depending on whether the frequency is positive or negative. This ends up in doubling the amplitude of positive frequencies and cancelling out negative frequencies.
    Then real sinusoids have become complex sinusoids.
    (Not that simple if you stop sticking to $ \cos $, $ \sin $ and $ \exp $.)

    A very convenient tool indeed. But I'd rather find a way to study signals without transforming them (even though it's easily reversible).
    So I'll take a different route.

    If all partials have their frequencies related by an integer factor, they are called harmonics. Then the one with the lowest frequency is called fundamental and all the other ones are called overtones.

    $ \alpha [ n ] \ $ means that each partial has its own amplitude without necessarily being mathematically related to n.

    DC offset

    Now, we just have to check what happens to negative-frequency partials once multiplied with our normalising sinusoid:

    $$ \begin{align} & \alpha (\omega) \cos \bigl( \omega t + \varphi (\omega) \bigr) \exp (-j \omega_p t) \ = \ \alpha (\omega) \frac { \exp \bigl( j \omega t + j \varphi (\omega) \bigr) + \exp \bigl( -j \omega t - j \varphi (\omega) \bigr) }{2} \exp (-j \omega_p t) \\ \\ & = \ \frac{\alpha (\omega)}{2} \exp \bigl( j \omega t + j \varphi (\omega) - j \omega_p t \bigr) \ + \ \frac{\alpha (\omega)}{2} \exp \bigl( -j \omega t - j \varphi (\omega) - j \omega_p t \bigr) \\ \\ & = \ \frac{\alpha (\omega)}{2} \exp \bigl( j \varphi (\omega) \bigr) \exp \bigl( j (\omega - \omega_p) t \bigr) \ + \ \frac{\alpha (\omega)}{2} \exp \bigl( -j \varphi (\omega) \bigr) \exp \bigl( -j (\omega - \omega_p) t \bigr) \end{align} $$

    The frequencies become: $ \ \omega - \omega_p \ $ and $ \ - \omega - \omega_p \ $.

    All these terms are integer multiples of $ d\omega $, so their partials will all cancel to zero once we integrate over time — except if $ \ \omega = \omega_p \ $ — then the negative frequency doubles (still integer multiple, alles gut); and the positive one becomes zero.

    Only $ \ X (\omega_p) \ $ is left.

    Notice (or remember) that the negative-frequency part of cosine has an opposite phase — cosine is the sum of a complex conjugate pair.
    This explains why it's real:

    $$ z + \conj (z) \ = \ \Re (z) + j \Im (z) + \Re (z) - j \Im (z) \ = \ 2 \Re (z) $$

    So if we want to split our cosine and go bilateral, we need to address this matter.

    We'll need the signum (or sign or sgn) function: 

        sign (x)  =  (x == 0.0)  ?  0  :  x / abs (x);

    that is:  zero if $ x $ is null, $ - 1 $ if it's negative, $ + 1 $ if it's positive
    (assuming $ x $ is real, otherwise it normalises its magnitude to 1 and places it on the unit-circle).

    Then, since:

    $$ x \ \geq \ 0 \quad \iff \quad \ \lvert x \rvert \ = \ x \qquad \AND \qquad x \ \lt \ 0 \quad \iff \quad \lvert x \rvert \ = \ - x $$

    if we substitute:

    $$ \varphi (\omega) \quad \longleftarrow \quad \varphi ( \lvert \omega \rvert ) \sgn (\omega) $$

    we get:

    $$ \omega \ \gt \ 0 \quad \implies \quad j \omega t + \varphi ( \lvert \omega \rvert ) \sgn (\omega) \ = \ j \omega t + \varphi (\omega) $$

    and:

    $$ \omega \ \lt \ 0 \quad \implies \quad j \omega t + \varphi ( \lvert \omega \rvert ) \sgn (\omega) \ = \ j \ t - \varphi (- \omega) $$

    and:

    $$ \omega \ = \ 0 \quad \implies \quad j \omega t + \varphi ( \lvert \omega \rvert ) \sgn (\omega) \ = \ 0 $$

    Finally: $ \ \alpha (- \omega) = \alpha (\omega) \ $, so we can use $ \ \alpha ( \lvert \omega \rvert ) \ $ for both the positive and negative component.
    Perfect.

    Go.

    $$ X (\omega_p) \ = \ \int\limits_{- \pi / d\omega}^{+ \pi / d\omega} \left( \int\limits_{- \infty}^{+ \infty} \frac{\alpha ( \lvert \omega \rvert )}{2} \exp \bigl( \ j \varphi ( \lvert \omega \rvert ) \sgn (\omega) \bigr) \exp \bigl( \ j (\omega - \omega_p) t \bigr) \frac{d\omega}{2 \pi} \right) \ dt $$

    Meet the Fourier transform (not inverse this time).

    You probably won't encounter this way of expressing it elsewhere, but we've built it together: you know it's valid.

    (Shall I dare call it the Working-class Fourier transform?)

    It doesn't have one single consensual expression because the division by $ 2 \pi $ (being a constant) can be placed anywhere.

    We've left it inside the inverse transform ( $ f (t) \ $, everything inside parenthesis ), that's one possibility; we'll leave it exactly that way to keep a trace of the original $ T_0 $ visible.

    It's often left with the inverse transform but moved outside of its integral.

    It can be transferred to the (not inverse) transform, or split in half:

    $$ \frac{1}{2 \pi} \ = \ \frac{1}{\sqrt{2 \pi}} \cdot \frac{1}{\sqrt{2 \pi}} $$

    and equally shared by each transform (inverse and not inverse)

    for the sake of symmetry.

    (I get the appeal: if your whole ecosystem is symmetrical, you only have to process half of it.
    Obsessing over palindromes can nourish art — ask Anton Webern; or even sociology — ask Guy Debord.)

    Of course, no one bothers with absolute values and the $ \sgn $ function: the phase is quietly assumed automatically reversed.

    In fact, you rarely encounter both integrals together like this: $ \ X (\omega_p) \ $ integrates $ \ f (t) \exp (-j \omega_p t) \ $ over time, and $ \ f (t) \ $ integrates $ \ X (\omega_p) \ $ over frequency — $ \ \alpha (\omega) \ $ and $ \ \varphi (\omega) \ $ being embedded in $ \ X (\omega_p) \ $.

    Finally, since $ \ \pi / d\omega \ $ tends to infinity when $ d\omega $ tends to zero (which it should, by definition of infinitesimal), the range of the temporal integration is classically simplified as $ \ [ \ - \infty \ , \ + \infty \ ] \ $ — which gives a beautiful symmetry to the nested integrals, but erases the record of how we got there (by stretching the Fourier series). Plus, free usage of the differential as a regular number is highly non-conventional — that's where we diverge the most.

    08 — ἀρχαί

    Let's go full on first principles.

    Change

    Let $ \ f (x) \ $ be some function of some variable $ x $; and $ a $ and $ b $ be any values of $ x $ (they can be equal if they will).
    Let's agree to stick to the largest set of numbers we need: complex numbers (which include real numbers, positive integers, etc.).

    When $ x $ evolves from $ a $ to $ b $ (I didn't write increase because it could very well be a decrease or a stagnation) — it goes from $ \ x = a \ $ to $ \ x = b \ $.

    Since: $ \quad b \ = \ b + a - a \ = \ a + (b - a) \ $,
    going from $ a $ to $ b $ is adding $ \ (b - a) \ $ to $ a $.

    Then $ \ (b - a) \ $ is the change of $ x $ between $ a $ and $ b $. Let's call it $ dx $.

    Then:

    $$ b \ = \ a + dx $$

    So: $ \quad f (b) \ = \ f (a + dx) $

    Which means that, when $ x $ changes from $ a $ to $ b $, $ \ f (x) \ $ changes from $ \ f (a) \ $ to $ \ f (b) \ = \ f (a + dx) \ $.

    The difference between $ \ f (b) \ $ and $ \ f (a) \ $ is:

    $$ f (b) - f (a) \ = \ f (a + dx) - f (a) $$

    That's the change of $ \ f (x) \ $ when $ x $ goes from $ a $ to $ b $. Let's call it $ df $.

    Since $ a $ and $ b $ can be anything, we could decide that: $ \quad a = x \ $.

    Then $ dx $ would express the change of $ x $ from any value;
    and $ \ f (x + dx) - f (x) \ $ would be the corresponding change of $ \ f (x) \ $.

    Rate

    The Latin verb reor, reri whose past participle is ratus, -a, -um, means to reckon, to compute, to think, to judge, ... — basically: to use a brain.

    It gave ratio: reason (sometimes loosely confused with cause), which passed to English and German with the meaning of arithmetical division.
    Reor (or rather its participle ratus, -a, -um) also gave rate which describes a proportional relationship between two things.

    E.g.:   if there are 1,000,000 flat-earthers amongst 8,000,000,000 Earthers:

    $$ \frac{1,000,000}{8,000,000,000} \ = \ \frac{1}{8,000} \ = \ 1/8 \ \E -3 \ = \ 0.125 \times 0.001 \ = \ 0.000125 $$

    is the rate of flat-earthers.

    Times 100 gives you a percentage: $ \quad 0.0125 \ \% \ $.

    Or, if you've walked 1 km in ten minutes:

    $$ \frac{1,000}{600} \ = \ \frac{10}{6} \ = \ 1 + 2/3 \ \metre / \second $$

    is your average walking rate.

    (A frequency — cycles or radians per second — is therefore a rate. Just like any speed.)

    Rate of change

    If we want to know how much $ \ f (x) \ $ changes in proportion of a corresponding change in $ x $,
    we'll get:

    $$ \frac{df}{dx} \ = \ \frac{ f (x + dx) - f (x) }{dx} $$ The ratio of changes, commonly referred to as rate of change or first derivative.

    Except that this formula implies that $ \ df / dx \ $ is constant:

    $$ \frac{df}{dx} \ = \ k \ \ \iff \ \ \frac{f (x + dx) - f (x)}{dx} \ = \ k \ \ \iff \ \ f (x + dx) \ = \ k \cdot dx + f (x) $$

    If we know the initial value:

    $$ f (0) \ = \ c \ \ \implies \ \ f (0 + dx) \ = \ k \cdot dx + f (0) \ = \ k \cdot dx + c $$

    Since $ dx $ can be anything, we've just found the generic expression of functions with a constant rate of change:

    $$ f (x) \ = \ k x + f (0) $$

    where $ k $ is constant (notice that nothing prevents any one of the terms from being zero).

    These functions are called linear because they're the equations of straight lines.

    If $ k $ is non-zero, a linear function is called affine. Both terms are very frequently used interchangeably.

    Although, if $ k $ and $ c $ are null, $ \ f ( x ) \ = \ 0 \ $ is a straight line (the abscissa axis), yet the function is not affine per se.
    (We'll properly deal with this terminology when discussing polynomials in Chapter v: Poles and zeroes .)

    Calculus

    The big idea of Calculus is to pick two points on a non-linear function, assume it's linear between these two points, repeat at fixed horizontal intervals to get an approximation; then reduce the interval as much as possible to get a more precise approximation; the ultimate limit being zero.

    But dividing by $ dx $ when $ dx $ equals zero won't help, so we explicitly express it as a limit.

    (We're sticking to Riemann integrals for now, so continuity is a requirement of integration until further notice — real numbers as well.)

    When I write assume it's linear between these two points, linear could have two meanings: constant or affine.
    Affine leads to trapezoidal integration (which will have to wait for Chapter vii: Ones and zeroes .)
    Constant means we build our functions with successive flat steps in a staircase fashion (which is the Leibnizian standard).

    E.g.:

    $$ f (x) \ = \ \frac{1}{x} \ \ \implies \ \ \frac{df}{dx} \ = \ \lim_{dx \to 0} \frac{ \dfrac{1}{x + dx} - \dfrac{1}{x} }{ dx } \ = \ \lim_{dx \to 0} \frac{ \dfrac{x - x - dx}{x^2 + x \cdot dx} }{ dx } \ = \ \lim_{dx \to 0} \frac{ - \dfrac{dx}{dx} }{x^2 + x \cdot dx} \ = \ \lim_{dx \to 0} \frac{- 1}{x^2 + x \cdot dx} $$

    As $ dx $ tends to zero, so does $ \ x \cdot dx \ $, so $ \ x^2 + x \cdot dx \ $ tends to $ x^2 $, so: $ \quad df / dx \ = \ -1 / x^2 \ $.
    This rate of change is always negative (if $ x $ is real) so the function is always decaying.

    Look at that: the function is undefined at zero, its rate of change is symmetrical around it...

    If the curve is your referential (meaning you travel on it and see everything else moving) — you'll jump from minus infinity to plus infinity without noticing: your acceleration will remain smoothly continuous.

    (You may experience an infinitesimal blackout... but maybe it just happened, how would you know?)

    As if real numbers were on a circumference where minus infinity and plus infinity were the same point.

    Pretty much like if you're on a moon of a moon of a moon of a moon of a planet of a star, and look at the sky, everything revolves around you — short-sighted frames of reference make the marvellous trivial.

    Now what if:

    $$ X \ = \ \int\limits_{a}^{b} f (x) \ dx \quad \text{?} $$
    4 cases:
    1. $ a $ and $ b $ are constant: $ X $ is constant and its rate of change is null;
    2. $ a $ is constant, $ b $ is not: $ X $ is a function of $ b $ and should be written $ \ X (b) \ $;
    3. $ b $ is constant, $ a $ is not: $ X $ is a function of $ a $ and should be written $ \ X (a) \ $;
    4. $ a $ and $ b $ are varying: $ X $ is a function of both and should be written $ \ X (a , b) \ $.

    Let's go with case (2) $ a $ constant; $ b $ variable; $ \ X \ = \ X (b) \ $. :

    $$ X (b) \ = \ \int\limits_{a}^{b} f (x) \ dx \quad \implies \quad \frac{dX}{db} \ = \ \lim_{db \to 0} \pfrac{ \displaystyle\int\limits_{a}^{a + db} f (x) \ dx \ - \ \displaystyle\int\limits_{a}^{b} f (x) \ dx }{ db } \ = \ \lim_{db \to 0} \left( \int\limits_{b}^{b + db} f (x) \ \dfrac{dx}{db} \ + \ \int\limits_{a}^{b} f (x) \ \dfrac{dx}{db} \ - \ \int\limits_{a}^{b} f (x) \ \dfrac{dx}{db} \right) $$ $$ \dfrac{dX}{db} \ = \lim_{db \to 0} \int\limits_{b}^{b + db} f (x) \ \dfrac{dx}{db} $$

    I can think of three paths by which we can go from here.

    Let's try all three.

    First path (the Clean):

    We're dividing the integral by: $ \ db \ = \ b + db - b \ = \SPAN \ $.
    So we're computing the average of all the values between $ \ f (b) \ $ and $ \ f (b + db) \ $.

    As $ db $ tends to zero, $ \ b + db \ $ tends to $ b $, and $ \ f (b + db) \ $ tends to $ \ f (b) \ $.

    So the limit case is the average value of a constant, ergo:

    $$ dX/db \ = \ f (b) $$

    Second path (the Unorthodox):

    $$ \dfrac{dX}{db} \ = \lim_{db \to 0} \int\limits_{b}^{b + db} f (x) \ \dfrac{dx}{db} $$
    Induction time!

    3 possibilities:

    1. $ db \lt dx $
    2. $ db \gt dx $
    3. $ db = dx $

    Possibility (1): $ \ db \lt dx $
    Then: $ \ b + dx \ \gt \ b + db \ $.
    The integral exceeds its prescribed range. Possibility (1) wasn't a possibility.

    Possibility (2): $ \ db \gt dx $
    Then $ db $ could be smaller.
    The limit is not a limit. Possibility (2) wasn't a possibility.

    Possibility (3): $ \ db = dx $ Then: $ \ b + dx \ = \ b + db \ $. The integral has only one step:

    $$ \dfrac{dX}{db} \ = \ \lim_{db \to 0} \int\limits{b}{b + db} f (x) \ \dfrac{dx}{db} \ = \ \lim_{db \to 0 \\ db = dx} f (b + db) \ \dfrac{dx}{db} \ = \ \lim_{db \to 0} f (b + db) \ = \ f (b) $$

    Why this is unorthodox: seeking two limits at once could be thought of as equivalent to implying that: $ \ 0/0 = 1 \ $.

    Which tends to hold for odd functions in Calculus, where limits and rates of change and limits of rates of change work around the aporia.
    (Take $ \ x/x \ $ or $ \ \sin (x) / \sin (x) \ $ or whatever over the same whatever — the curve is 1 everywhere and undefined when whatever is zero.
    But since it's 1 before and after, the rate of change is zero everywhere, and as you tend to 0/0, no rate of change can make the ratio escape 1.

    This, though, relies on a rather bold petitio principii: constant rational functions have no domain discontinuity...

    A principle every one is free to reject or admit.

    Even functions, like $ \ x/x^2 \ $ break this tendency — as well as some asymmetrical functions like $ \ \bigl( \exp (x) - 1 \bigr) / 4 x^2 \ $.)

    Whereas, in algebra, $ 0/0 $ is always multi-valued (and therefore undefined): $ \quad x \cdot 0 \ = \ 0 \quad \iff \quad 0 / 0 \ = \ x $ i.e. literally anything — pretty similarly to infinity.
