    11 — Alternative pre-warping

    Disclaimer: I didn't come up with the following idea.
    It's Robert Bristow-Johnson's technique — and it's exposed at the end of his famous cookbook.
    That is: after the copy/paste-ready closed formulae, RBJ shows us where they come from: not black magic, only Calculus and trigonometry — ok, maybe that's what black magic refers to.
    This means: his cookbook didn't have to be one — had people read it in full.
    (As you can see, I didn't come up with the pseudo-code monospace aesthetic (... GitHub's the original
    — you're reading the derivative...)     either — opposite of a cookbook... more like Through the Looking-Glass.)
    $ \bigl( $( Monospace block diagrams Is this field mandatory? are 100% Working-class block diagrams though.)$ \bigr) $
    Robert injects BLT into the transfer function in unit-cutoff normalisation.
    So the substitution is not operated on $ s $, but rather on $ \ s' \ = \ s / \omega_c \ $.
    For it to work properly, we need $ s $ and $ \omega_c $ to be commensurable.
    We need to rescale $ \ 0 \ \lt \ {\omega_c T} \ \lt \ \Nyquist \ $ as $ \ 0 \ \lt \ \omega_c \ \lt \ + \infty \ $.
    So:
    $$ \lim_{\omega_c \ \to \ \pi / T} \ \longleftarrow \ \lim_{\omega_c \ \to \ + \infty} $$
    We've already found the formula (#05 and (#06):
    $$ \omega_c \ \longleftarrow \ 2 \tan ( \omega_c T / 2 ) / T $$
    In case you're reading RBJ's cookbook in parallel and wonder why there is no $ T $ in his formula:
    we're using $ \omega_c $, he's using w0 — which is defined around the beginning (Then compute a few intermediate variables), as: w0 = 2*pi*f0/Fs which here corresponds to:
    $$ 2 \pi \f_c / \fs \ = \ \omega_c / \fs \ = \ \omega_c T $$
    then w0/2 corresponds to:
    $$ \omega_c T/2 $$
    alles gut.
    The substitution:
    $$ s' \quad \longleftarrow \quad \pfrac{ \dfrac{2}{T} \cdot \dfrac{1 - z^{-1}}{1 + z^{-1}} \rule[-2.5ex]{0pt}{0pt} }{ \rule[4ex]{0pt}{0pt} \dfrac{2}{T} \tan (\omega_c T/2) } \ = \ \frac{1}{\tan (\omega_c T/2)} \cdot \frac{1 - z^{-1}}{1 + z^{-1}} $$
    Ok, clean. Except the function $ \ \tan (x) \ $ tends to infinity when $ x $ tends to $ \pi / 2 $,
    so $ \ 1 / \tan (\omega_c T/2) \ $ will give very bad approximations when the frequency tends to $ \Nyquist $.
    Luckily there are legions of trigonometric identities. So we can likely re-express $ \ 1 / \tan (x/2) \ $ in terms of cosine and sine.
    Except...
    $$ \tan (x) \ = \ \pfrac{\sin (x)}{\cos (x)} $$
    so
    $$ \frac{1}{\tan (x)} \ = \ \frac{\cos (x)}{\sin (x)} $$
    and $ \ \sin (x) \ = \ 0 \ $ means also very bad approximations when the frequency tends to DC.
    So the ideal solution should only rely on cosine.
    Easy: $ \ \sin (x) \ = \ \cos (x - \pi / 2) \ $ ...  which doesn't fix the DC issue.
    Looking back at RBJ's solution, he didn't bother with the sine issue.
    I guess when people accuse traditional BLT of quantisation noise at DC, they do in fact know their shit...
    Or at least they trust people who do.
    Edit:  As we're about to find out, it's more complicated than that (spoiler: denominators can cancel out...).

    $$ \frac{1}{\tan (x/2)} \ = \ \frac{\cos (x/2)}{\sin (x/2)} $$

    Let's be bold and try solving both equations at once, using: $ \ \exp (j x) \ = \ \cos (x) + j \sin (x) \ = \ \zeta \ $.
    So:
    $$ \zeta \ = \ \exp (j 2 \xi) \ = \ \exp (j \xi)^2 \quad \iff \quad \cos (2 \xi) + j \sin (2 \xi) \ = \ \bigl( \cos (\xi) + j \sin (\xi) \bigr)^2 \\ \\ \iff \quad \cos (2 \xi) + j \sin (2 \xi) \ = \ \cos^2 (\xi) + j 2 \cos (\xi) \sin (\xi) - \sin^2 (\xi) $$
    so:
    $$ \begin{alignat}{3} \Re (\zeta) \ = \ &\cos (2 \xi) \ &= \ &\cos^2 (\xi) - \sin^2 (\xi) \\ \Im (\zeta) \ = \ &\sin (2 \xi) \ &= \ &2 \cos (\xi) \sin (\xi) \end{alignat} $$
    Then:
    $$ - \sin^2 (x/2) \ = \ \cos (x) - \cos^2 (x/2) \quad \iff \quad \sin^2 (x/2) \ = \ \cos^2 (x/2) - \cos (x) \\ \\ \iff \quad 1 - \cos^2 (x/2) \ = \ \cos^2 (x/2) - \cos (x) \quad \iff \quad 1 - 2 \cos^2 (x/2) \ = \ - \cos (x) \\ \\ \iff \quad 2 \cos^2 (x/2) \ = \ \cos (x) + 1 \quad \iff \quad \cos (x/2) \ = \ \pm \sqrt{ \frac{\cos (x) + 1}{2} } $$
    and:
    $$ \cos^2 (x/2) \ = \ \cos (x) + \sin^2 (x/2) \quad \iff \quad 1 - \sin^2 (x/2) \ = \ \cos (x) + \sin^2 (x/2) \\ \\ \iff \quad 2 \sin^2 (x/2) \ = \ 1 - \cos (x) \quad \iff \quad \sin (x/2) \ = \ \pm \sqrt{ \frac{1 - \cos (x)}{2} } $$

    This can look messy at first sight.
    But many recurring terms can be pre-computed.
    Cosine and sine can be set first.
    Then $ \ 1 - \cos \ $ and $ \ 1 + \cos \ $.
    I've normalised $ \alpha_0 $ to $ 1 $,
    so all coefficients share the same denominator...
    The analog prototype's transfer function uses the unit-cutoff normalisation,
    so there's a strong probability that several analog coefficients will be equal to $ 1 $.
    The $ \alpha $ coefficients have common terms in their numerator and in their denominator — they can be pre-computed before setting the denominator.
    For LP filters, you'll likely have $ \ b_0 = 1 \ $ and $ \ b_1 = b_2 = 0 \ $. Etc.
    Looking back at RBJ's cookbook, it seems that he divided the numerator and the denominator by 2 — probably to get rid of the factor 2 in $ \beta_1 $ and $ \alpha_1 $, and to avoid a factor 4 in $ \alpha_1 $, due to the topology he's implementing.
    This is pure normalisation: it doesn't break the equality (perhaps better approximation?).
